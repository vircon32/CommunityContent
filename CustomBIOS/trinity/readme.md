# TRINITY BIOS

An alternative BIOS for Vircon 32, with rich animations and visual effects.

![amazing screenshot](https://i.postimg.cc/15SZbpBn/2026-02-15-15-50-35.png)
<br/><br/>

---

## INSTALATION:

### 1. Download the BIOS.
 Get the trinity.v32 file found in the build directory.

### 2. Move the bios.
 Copy trinity.v32 file into the Bios directory found in your emulator folder.

### 3. Configure the emulator.
 Open Config-Settings.xml file with a text editor and change the bios file name to "trinity.v32"
