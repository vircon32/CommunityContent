# TWO SIDES OF A BUNN

A visual demostration showing a not CPU intensive method to draw cropped textures, this demo features two totally different bunny textures, the **Awesome bunny** will only be drawn behind the division line, and the **Good Bunny** will only be drawn after the division line.
